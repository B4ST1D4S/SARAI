IMPORTANTE — PALABRA DE ACTIVACIÓN
────────────────────────────────────────────────────────────────────────────────
Todos los comandos del MODO COMANDOS (SpeechRecognition) deben comenzar con:

   "Sarai ..."

El sistema reconoce variantes fonéticas del STT en español:
  Sarai / Sara / Saray / Sarah / Sarahi / Zarai / Sary / Sari / Sarahy
  o precedido de: hey, oye, hola  →  "Hey Sarai ...", "Oye Sarai ..."

Hay DOS canales de entrada:
  [W] = Vía WHISPER  (grabar audio → se transcribe automáticamente)
  [S] = Vía SPEECH   (SpeechRecognition continuo, ya activado)
  [T] = Vía TEXTO    (escribir en el campo y pulsar Enter)

6. RESUMEN RÁPIDO DE FLUJO
================================================================================

  PASO 1 → Clic en "Iniciar grabación" (único clic obligatorio)
  PASO 2 → Di "activar comandos"  →  SpeechRecognition arranca solo
  PASO 3 → A partir de aquí, todo por voz:

    "Sarai ir a pacientes"          → navega a Pacientes
    "Sarai abrir cotizaciones"      → navega a Cotizaciones
    "Sarai central impresion"       → navega a Central Impresión
    "Sarai usuarios"                → navega a Gestión de Usuarios
    "Sarai manual"                  → abre Manual de Usuario
    "Sarai grabar"                  → inicia grabación Whisper
    "Sarai parar"                   → detiene grabación y transcribe
    "Sarai nueva historia"          → abre formulario HC
    "Sarai motivo de consulta"      → va a esa sección del formulario
    "Sarai desactivar comandos"     → apaga SpeechRecognition


================================================================================
8. NOTAS TÉCNICAS
================================================================================

  • El sistema normaliza el texto: elimina tildes, convierte a minúsculas.
    "Pacientes" / "PACIENTES" / "pácientes" se reconocen igual.

  • SpeechRecognition usa lang='es' (cubre todos los dialectos hispanohablantes).

  • Whisper usa modelo 'medium' — reconoce con alta precisión incluso con
    acento colombiano u otros dialectos.

  • Si el micrófono está bloqueado en Chrome:
      → Clic en el candado 🔒 junto a la URL
      → Micrófono → Permitir
      → Recargar la página (F5)

  • El campo de texto (fallback) acepta cualquier comando aunque el micrófono
    esté deshabilitado.
