# 🎤 MÓDULO CONSULTA EXTERNA - Whisper + Agenda + Seguimiento

## ¿QUÉ INCLUYE?

Enfoque **SOLO** en atención de consultorios de medicina estética externa:
- Whisper (dictado por voz)
- Control y seguimiento de agenda
- Gestión de pacientes en consulta
- Consentimiento digital
- Módulo visual básico
- Cotizaciones rápidas

## 🎤 COMANDOS DE VOZ (Whisper)

Dicta directamente durante la consulta:

| Comando | Resultado |
|---------|-----------|
| "Paciente refiere..." | Guarda observación |
| "Alergia a penicilina" | Agrega a alergias |
| "Próxima cita en 2 semanas" | Programa seguimiento |
| "Cotización: botox + relleno" | Genera presupuesto |
| "Foto antes" | Marca foto como "antes" |
| "Foto después" | Marca foto como "después" |
